# ☁️ Cloud Learning & DevOps Projects

This repository contains a collection of hands-on projects focused on **Docker, Docker Compose, Cloud Concepts, and DevOps practices**.  
It demonstrates real-world implementation of containerization, multi-service architecture, environment management, and auto-scaling systems.

---

## 📌 About the Repository

This repo is a structured learning journey covering:

- 🐳 Docker fundamentals
- 🔗 Docker Compose (multi-container apps)
- ⚙️ Environment variable management
- 📊 Monitoring & auto-scaling system
- 🗄️ Database integration using MySQL
- 🏗️ Microservices-based architecture

Each project is designed to simulate real-world DevOps scenarios.

---

## 🧠 Key Concepts Covered

### 🐳 Docker
- Creating custom images using Dockerfile
- Container lifecycle management
- Port mapping and volume mounting

### 🔗 Docker Compose
- Multi-container application setup
- Backend + Frontend + Database integration
- Service communication

### 🌱 Environment Variables
- Managing `.env` files
- Secure configuration handling
- Dynamic app configuration

### 📊 Monitoring & Auto Scaling
- Prometheus for metrics collection
- Grafana dashboards for visualization
- Nginx as load balancer
- Auto-scaling using scripts

### 🗄️ Database (MySQL)
- Containerized MySQL setup
- Initialization scripts
- Persistent storage

## ▶️ How to Run Projects

### 1️⃣ Clone the repository
git clone https://github.com/ayishwaryaC/Cloud-learning.git
cd Cloud-learning

### 2️⃣ Run any project
Example:
cd "docker compose projects/Docker compose project 2"
docker compose up -d

### 3️⃣ Access application
Frontend → http://localhost:<port>
Backend → http://localhost:<port>
